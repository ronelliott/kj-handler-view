## Installation
Install using [NPM](https://github.com/isaacs/npm):

    npm install kj-handler-view --save


## Usage
Add the module to the modules section of your config:

    modules.exports = {
        ...
        modules: [
            ...
            require('kj-handler-view'),
            ...
        ],
    };
